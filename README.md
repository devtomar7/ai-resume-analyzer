# AI Resume Analyzer

AI Resume Analyzer built using Python and Flask that evaluates resumes and provides ATS feedback.

## Features
- Resume Score
- ATS Match Percentage
- Skill Detection
- Recommended Role
- Missing Skills
- AI Resume Feedback

## Tech Stack
- Python
- Flask
- HTML/CSS
- PyPDF2
- ReportLab

## Live Demo
https://ai-resume-analyzer-xorj.onrender.com

## Run Locally
pip install -r requirements.txt
python app.py
